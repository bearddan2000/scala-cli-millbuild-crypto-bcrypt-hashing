# scala-cli-millbuild-crypto-bcrypt-hashing

## Description
A demo for Bcrypt one-way hashing of
a password.

## Tech stack
- scala
- millbuild

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
